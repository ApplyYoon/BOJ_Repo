# (BOJ) 백준 9084 - 동전
소요시간: 1h 30m

## 풀이
### 핵심(풀이) 키워드
- Dynamic Programing

### 아이디어
- 목표 금액 M을 만드는 경우의 수를 dp[i]라고 정의
- 초기값: dp[0] = 1 (아무 동전도 사용하지 않는 1가지 방법)
- 각 동적 C에 대해 dp[i] += dp[i-c]

예) 마지막에 5원을 쓴 경우
- 10원을 만들려면 맨 끝에 5원을 하나 둔다.
- 그러면 그 앞은 10-5=5원을 만드는 방법이 되어야 한다.
- 즉, dp[10] = dp[10] + dp[5]  (dp[10] += dp[5])
